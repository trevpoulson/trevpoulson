Introduction

Problem

AAPC was starting a new sister company, Healthicity, and needed to create a visual identity that was different from it's current brand.

Starting Design

To start off, There were multiple products and marketing sites that ran the gambit of visual design. We needed to create a visual identity that could exist within marketing design and unify the visuals of the products.

Design

There were multiple products and websites and we wanted a single brand that would look modern and feel like a modern product company.

For visual styling, we leaned heavily on the Material Design language created by Google but made it our own by choosing custom colors, layouts and fonts that worked better for our brand. In the end, we created a style and brand that was unique to Healthicity and let our customers know what we stood for.

Perfection is achieved, not when there is nothing left to add, but when there is nothing left to remove.

Brand Message and Approach

Healthicity is about simplicity. We strive to incorporate simplicity into everything we do. From marketing to product design, It is an integral part of our brand.

We chose simple fonts, imagery, iconography and color pallets to illustrate simplicity. Our content and messaging is at the forefront of our designs while the styling appears to be invisible because of it’s simplicity. This isn't to say that we are not focused on design. On the contrary, we spend a great deal of time working on the design of everything we create. We sweat over every detail, striving for perfection through simplicity. We ask ourselves “is this necessary” for every component and aspect of our designs, while stripping away everything that isn't essential.

Style Guides

The first step was to create a pattern library and style guide so we could maintain a consistent brand image for marketing and code style sheets for product.

Style Guide

Initially, we made style guide that outlines font sizes, weights, colors, and logos for use by internal and external teams.

Pattern Library

I created the pattern library using variable based LESS and Angular in a way that any of the applications, including the marketing website could use the same code and only have to change a few variables. Furthermore, updates could be made across the entire company with a change to only one place. You can see the Pattern Library Here. It is, and always will be a work in progress because our products are always changing.

Marketing

We've created numerous marketing materials, from trade-show booths, to marketing websites and even powerpoint templates that all use the style guide layout. You can see the website at www.healthicity.com if you'd like to see more.

Product

We have 5 products that are in production or active development that are using the Pattern Library. Designs are created using sketch and a static version of the pattern library, then converted to code for prototyping using the actual pattern library.
